# Sanity Next.js Development Skill - Quick Start

## Installation

1. **Download** the `sanity-nextjs-dev.zip` file
2. **Go to Claude.ai** → Settings → Skills
3. **Click** "Add Skill" → "Upload Skill"
4. **Select** the downloaded ZIP file
5. **Enable** the Skill for your project

## First Test

Try this prompt to verify it's working:

```
Set up a new Next.js 15 project with Sanity CMS. Include TypeScript and configure it for an embedded Studio.
```

You should get complete setup instructions, code examples, and configuration files.

## What This Skill Does

Transforms Claude into an expert Sanity.io + Next.js developer that helps you:

- ✅ Set up projects with best practices
- ✅ Design robust, validated schemas
- ✅ Write type-safe GROQ queries
- ✅ Implement visual editing & live preview
- ✅ Integrate with Supabase, OpenRouter, Vercel AI SDK
- ✅ Deploy to Vercel with webhooks
- ✅ Optimize performance
- ✅ Troubleshoot issues

## Tech Stack Covered

- **Sanity.io** (v3+) - Headless CMS
- **Next.js 15** - App Router with React Server Components
- **TypeScript** - Full type safety
- **GROQ** - Query language with TypeGen
- **Supabase** - Database and auth
- **OpenRouter** - AI model gateway
- **Vercel AI SDK** - Streaming AI responses
- **Vercel** - Deployment platform
